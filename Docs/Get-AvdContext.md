# Get-AvdContext

## SYNOPSIS
Get the current context

## SYNTAX

```
Get-AvdContext [<CommonParameters>]
```

## DESCRIPTION
In the case you need to set a new subscription context, use this command Set-AvdContext -subscriptionId \<xxxx-xxxx\>.

## EXAMPLES

### EXAMPLE 1
```
Get-AvdContext
```
