# Disconnect-Avd

## SYNOPSIS
Disconnects from AVD

## SYNTAX

```
Disconnect-Avd [<CommonParameters>]
```

## DESCRIPTION
Disconnects from AVD enviroment by clearing all tokens.

## EXAMPLES

### EXAMPLE 1
```
Disconnect-Avd
```
